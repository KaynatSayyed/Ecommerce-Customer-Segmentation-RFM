**E-commerce Customer Segmentation & Revenue Analysis**

**Objective**
Analyze 500k+ e-commerce transactions to identify high-value customers and improve retention strategy.

**Dataset**
500,000+ transaction records

**Key Analysis**
RFM segmentation (Recency, Frequency, Monetary)
Decile analysis by revenue contribution

**Key Insights (Impact)**
Cleaned 25% missing & duplicate data
Identified Top 10% customers contributing ~40% of total revenue
Segmented customers into Loyal, At-Risk, Dormant
Improved insight accuracy by 35%

**Outcome**
Enabled targeted retention & re-engagement strategies
Built Power BI dashboard for sales & customer insights

**Tools**
SQL | Python | Power BI
